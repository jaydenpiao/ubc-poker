# UBC Poker Club Web Application

## [ubcpokerclub.com](https://www.ubcpokerclub.com)

## Tech Stack

- JavaScript
- HTML
- CSS
- React
- Tailwind CSS

## Deploy Changes

1. git add
2. git commit
3. git push
4. npm run build
5. npm run deploy
6. go to pages in github repo settings
7. add ubcpokerclub.com to custom domain and save
